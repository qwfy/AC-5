This is a collection of heavily documented Python files containing TensorFlow codes/models, suitable for learning.

Each .py file is a standalone module, and comes with a solarized-light-themed .html file for online reading.

The .py file is supposed to be executed interactively, hence is organized using "cells", which is supported by various tools, such as Spyder, PyCharm and Neovim with senter.
